通过小组合作完成一个分布式系统的编写，能够在同一局域网下完成网络数据的交互等 等操作，并且能够自行应用网络协议，设计独特系统应用功能最后形成可视化界面。   
   
环境：win10  
开发工具：VS  
语言：C#  
  
Kerberos是一种计算机网络授权协议，用来在非安全网络中，对个人通信以安全的手段进行身份认证。  
这是一个基于kerberos实现的小说阅读器。
